# Saltabo

A modern macOS app that redefines how you switch between apps.

Fast. Minimal. Fully customizable.

---

## ✨ Why Saltabo?

macOS app switching is powerful — but limited.

Saltabo gives you full control:

- ⚡ Faster switching between apps
- 🎯 Smarter window selection
- 🎨 Customizable UI & behavior
- 🧠 Built for power users

---

## 🚀 Features

- Replace default Cmd+Tab behavior
- Preview windows when switching
- Keyboard-first navigation
- Smooth, low-latency performance
- Clean and minimal UI

---

## 🖥️ Built for macOS

Saltabo is designed specifically for macOS:

- Native performance
- Optimized for Apple Silicon
- Works seamlessly with your workflow
